## 요약

- 3가지 메뉴(병상현황, 응급메시지, 중증질환)의 주요 미구현 사항을 모두 구현 완료
- `get-severe-acceptance` API 추가 (HPID+QN 또는 Q0+QN 방식)
- 연령/체중 제한 배지 표시 추가 (영유아 관련 질환)
- Google Analytics 및 네이버 사이트 인증 적용 (전역 메타/스크립트)
